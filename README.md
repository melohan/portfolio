## Setting up dev

### Prerequisites

- npm @8.18.0

### Clone the repository

```shell
git clone git@github.com:melohan/portfolio.git
cd portfolio
```

### Install dependencies

```shell
cd custom-portfolio/
npm i
```

### Run the project (dev)

```shell
npm run dev
```
